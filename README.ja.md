# グランブルーファンタジー ツール（gbf-tools）

[繁體中文](README.md) ｜ [English](README.en.md) ｜ **日本語**

[グランブルーファンタジー](https://game.granbluefantasy.jp/) Web 版向けのユーザースクリプト集です。UI 改善、ショートカットバー、リアルタイム翻訳、そして「戦闘終了／全滅／青箱ライン」などのスマホ通知。各スクリプトは**独立**しているので、必要なものだけ入れられます。

## クレジットと注意

メインスクリプトは [biuuu](https://gist.github.com/biuuu) 氏の [blhx.user.js](https://gist.github.com/biuuu/b5fca321fc232b79161095c71a26f43f)（スクロールバー／サイドバー非表示、救援コードのコピー、BGM 継続など）を改変したものです。本 fork は kv が拡張しました。原作者に感謝します。著作権上の懸念があれば Issue を立ててください。

## インストール

まずユーザースクリプトマネージャーを導入：[Tampermonkey](https://www.tampermonkey.net/)（推奨）／[Violentmonkey](https://violentmonkey.github.io/)／[Greasemonkey](https://www.greasespot.net/)。
入れたい `.user.js` を開いて **Raw** を押すとインストール確認が出ます。全スクリプトに `@updateURL` を付けているので、以後は自動更新されます。

## メイン：`core/gbf-tools.user.js`

| 機能 | 説明 |
|---|---|
| 🔇 スクロールバー非表示 | Webkit スクロールバーを除去 |
| 🚫 Mobage サイドバー非表示 | 左の Mobage ナビを隠す |
| 🗨️ チャット非表示 | ゲーム内チャットを隠す |
| 📋 救援／部屋番号コピー可 | 救援コード・部屋番号を選択コピー可能に |
| 🎵 BGM 継続 | タブ切替えでも BGM が止まらない |
| 💧 雫メニュー拡張 | 回数メニュー先頭に 15〜11 を追加、既定 15 |
| 📈 スキルLv 自動最大 | スキル強化を自動で最高Lvに |
| 📦 個数 自動半分 | 最大値の半分以上で最小の選択肢を自動選択（アーティファクト頁は除外） |

## ショートカットバー：`core/shortcut-bar-*.user.js`

**カスタムショートカット**（タイトル＋リンク）の浮動バー。押すと指定ページへ直行（GBF 内部パス `quest`、`party/index/0/npc/0` など、または任意のフル URL）。

- **見た目 2 種、どちらか一方**（機能は同じ）：`core/shortcut-bar-glass.user.js`＝すりガラス；`core/shortcut-bar-native.user.js`＝GBF 純正ボタン画像。`@name` が同じなので、もう片方を入れると見た目だけ差し替わり、設定は保持。
- **ドラッグ可**：左上の **⠿** ハンドルを掴んで任意の位置へ、離すと位置を記憶（ローカル）。画面外に出しても自動で戻る。
- **カテゴリ**：ショートカットにグループを設定可。複数グループで金色の「カテゴリ切替え」チップが出ます。
- **ホットキー**：各ショートカットに 1 キーを割当て可（入力欄では発火しない）。
- **表示トグル**：⚙ 編集モードで追加／編集／削除／非表示（畳むとハンドル＋⚙ のみ）。
- **既定はローカルのみ**：ショートカットはブラウザ保存、サーバー不要。

### デバイス間同期（任意・自前バックエンド）

スクリプト冒頭の `SYNC_API` / `SYNC_TOKEN`（自分のエンドポイント）を埋めると複数端末で共有できます。契約は最小：`GET` で前回保存の JSON を返す（無ければ `null`）、`PUT` で request body をそのまま保存、いずれも `Authorization: Bearer <SYNC_TOKEN>` で認証。Cloudflare Workers + KV の無料枠で実装可。

## リアルタイム翻訳：`core/gbf-translate.user.js`

GBF の**日本語 DOM テキスト**をその場で中国語に翻訳——スキル／武器効果、召喚石、クエスト＆ストーリー、各種メニュー。原文を置き換え、結果をキャッシュして節約、ページ遷移にも追従。**既定は Google 翻訳（無料・キー不要）で即使用可**、必要ならメニューから DeepL（要キー）に切替え。

> ⚠ 戦闘画面のボタン／ダメージ数字／立ち絵テキストは**スプライト画像**でテキストではないため翻訳不可。

## 通知スクリプト（`notify/kv/` と `notify/bark/`）

戦闘関連のスマホ通知。**通知チャンネル**で 2 フォルダに分割、**どちらか一方のフォルダから導入**（同じスクリプトを両チャンネル入れると二重通知になります）：

| フォルダ | チャンネル | 向き |
|---|---|---|
| `notify/kv/` | **kv 通知ハブ**（自前 `POST /api/notify`） | 自分の通知バックエンドを持つ人 |
| `notify/bark/` | **Bark**（`api.day.app` 直結） | [Bark App](https://bark.day.app/) 利用者 |

### 収録スクリプト

| スクリプト | 機能 | kv | bark |
|---|---|:--:|:--:|
| `gbf-done.user.js` | **戦闘終了**（リザルト／他者が討伐）＋**全滅**（自パーティ壊滅）通知 | ✅ | ✅ |
| `gbf-aobako-line.user.js` | **青箱ライン**：戦闘中に「貢献度 vs この本の青箱ライン」バーを表示、超えると ✅、「ライン超え／全滅」を通知可 | ✅ | ✅ |
| `gbf-genki-notify.user.js` | **元気回復**：探検隊の元気が満タンで通知（**スケジュール式**、ブラウザを閉じても届く） | ✅ | — |

> 元気は「満タン時刻を算出 → 自前スケジューラへ送信 → 時刻に発火」のスケジュール通知で、本質的に kv 自前構成です。Bark に「タイマーで・閉じていても発火」の同等機能が無いため `notify/kv/` のみ。

### 設定（導入後）

**Tampermonkey アイコン → 当該スクリプト** のメニューから操作、**コード編集は不要**：
- 🔑 token／key 設定（`notify/kv/` は kv 通知ハブ token、`notify/bark/` は Bark device key）
- 🔔/🔕 各通知を**個別にオン／オフ**（戦闘終了／全滅／ライン超え／全滅）
- ℹ️ 現在の状態を表示

### 青箱ラインバー

- マルチ戦に入ると 1 行バー表示：`本名 ｜ 貢 貢献度 ｜ 線 青箱ライン ｜ 判定`。見た目はショートカットバー純正スキンと同じ、ドラッグ可・テキスト選択可。
- **貢献度**は MVP パネルの自分の行から、**本名**は戦闘画面テキストの照合で取得。
- 誤検出／未検出時は ⚙ メニューで**本名を手動上書き**、または候補文字列を一覧表示。
- 青箱ラインは**ボス別に内蔵**し `確定／推定／青箱なし／データなし` を明記（多くは版で変動する有志推定値。主な出典：灰机wiki、wikiwiki「青箱ライン一覧」、神ゲー攻略）。

> ⚠ **公開リポジトリの安全**：token／key／Bark key は必ず**手元のユーザースクリプトマネージャー**内のみに記入（またはメニューで設定＝ローカル保存）。実値を含む版を**絶対にリポジトリへ commit しないでください**。
